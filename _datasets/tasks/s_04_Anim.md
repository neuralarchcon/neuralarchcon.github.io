#### Detailed description

- Participants are presented with individual objects and scenes (same stimuli as in the associative memory task) and must categorize each as animate or inanimate as quickly and accurately as possible using key presses. The task measures implicit memory effects by comparing response speed and accuracy for previously seen stimuli versus new ones.
